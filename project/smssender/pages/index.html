About:

A while ago I created an aplication that send bulk messages just to troll my girlfriend, spaming him inbox with messages. So, a two weeks ago i remembered that time and created the same idea but better.

How works:

Essentially you have three inputs:

  • - Phone Number;
  • - Message body;
  • - Number of messages to send;

Phone Number: the phone number from the person who receives the SMS.

Message body: message that you want to send.

Number of messages to send: how many messages that you want to send to the person.

Caution:

This application sends real messages, so depending on the carrier you can be charged for each SMS sent!
